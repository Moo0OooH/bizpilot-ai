# BizPilot AI - Customer Onboarding Checklist v1.0

## Purpose

Define the manual founder-led onboarding path for the first cleaning-business pilots.

## Before Setup

Collect:

```text
business name
owner name and email
city and service area
cleaning services
existing quote intake channels
logo URL if available
brand colors if available
preferred reply tone
FAQ and pricing boundaries
current follow-up process
```

## Setup Steps

```text
create or confirm owner account
create business tenant
set plan and status in /admin
configure Business Profile
configure Quote Setup
activate public quote link
place quote link in at least two channels
create or submit one safe test lead
review lead detail with owner
generate/copy AI draft manually
mark lead contacted manually
record onboarding notes in Founder CRM
```

## Owner Training

The owner must understand:

```text
AI drafts are reviewed by the owner
nothing sends automatically
BizPilot is not booking or invoicing software
missing details should be confirmed before quoting
follow-up is manual during pilot
```

## Go-Live Checklist

```text
public quote link works
quote submission creates tenant-scoped lead
dashboard lead queue works
lead detail shows summary/missing info/draft
copy controls work
no send action exists
support contact path is clear
next weekly check-in is scheduled
```
