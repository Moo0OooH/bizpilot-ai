# VerifGo QC Mobile

Expo React Native app for the VerifGo QC daily verification product.

This app is intentionally separate from BizPilot. It consumes shared constants,
copy, and validation from `packages/verifgo-shared`.

## First Screens

- Today
- History
- Vehicle
- Settings
